About Snack Tracker

Snack Tracker is a student-developed prototype created for the ITIH101B4 Innovation and Technology course. The project is designed to help students check and report campus vending machine status in real time.

The system allows users to view vending machine inventory, check whether products are stocked, low, or out of stock, and submit status updates through a simple reporting page.

Development Team

This prototype was designed and developed by Jupiter Group as part of a collaborative academic project. The team worked on project research, interface design, prototype development, testing, and documentation.

Project Purpose

The purpose of this project is to explore how a simple digital system can improve access to information on campus. By using student-submitted reports and a shared dashboard, Snack Tracker aims to reduce wasted trips to unavailable vending machines and support better campus service awareness.

Disclaimer
